配置模板
===

group.js脚本适用于Android端clash系列的覆写或者扩展配置，主要给订阅添加策略分组，同时在订阅的基础上添加自建节点。

clash.yaml为Android端clash系列的配置配置文件，按需修改配置文件中的订阅链接和自建节点配置

⚠️区别：使用group.js脚本不影响订阅的自动更新，而使用clash.yaml配置文件订阅链接需要手动更新

本配置为自己写的自用配置改成的模板，未必适合所有人，请按需修改使用。
