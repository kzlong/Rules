配置模板
===

**group.js**脚本适用于Android端clash系列的覆写或者扩展配置，主要给订阅添加策略分组，同时在订阅的基础上添加自建节点。



**clash.yaml**为Android端clash系列的配置配置文件，按需修改配置文件中的订阅链接和自建节点配置



⚠️区别：使用**group.js**脚本不影响订阅的自动更新，而使用**clash.yaml**配置文件订阅链接需要手动更新



**Script.js**配置为**Clash Verge Rev**的扩展配置脚本，也可以用于**mihomo party**、**clash-nyanpasu**等其他的覆写



本配置为自己写的自用配置改成的模板，未必适合所有人，请按需修改使用。
